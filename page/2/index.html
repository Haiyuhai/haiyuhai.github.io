她说她要去养狗

发表于 2016-10-26 | 更新于: 2016-10-26 | | 热度: ℃
字数统计: 898 | 阅读时长 ≈ 2

十一之前，我接到一个电话，她说她把店给盘出去了。我问为什么？她没有回答。

高一那年，因为家庭的原因，她上了第一学期，就没有再读了。我记得她是很愤怒的告诉父亲说她不想读了。那时我很小，不懂什么情况。

最后她出去打工了，在外面租着小小的房子，自己付房租，自己做饭。一个可以通过自己赚得生活费的心情，对于我一个还在上学的人很是羡慕，因为可以去买想要的东西，可以满足自己小小的欲望。甚至偶尔回来还可以给妹妹一些零花钱。

后来她放弃了纺织的工作，去了一家美容院。然后她学会了打理自己，学会了收拾自己，学会了化妆，然后染上了洁癖。

即使后面她结婚了，她也会时不时的去美容院去做美容。即使她有了孩子，她也会拜托我照看孩子，然后去美容院。

我曾问她，为什么这么忙还是会去美容院，她说不想成为黄脸婆。

暑假回去，看到她每天忙着店里的生意。皮肤变得发黄，甚至长了痘痘。我说怎么不去美容院了，她说哪有时间，有时间也没有心去了。这时我对她讲，你可千万不要成为黄脸婆。

转眼，她的孩子已经四岁，并且上了幼儿园。我经常跟她讲，她在上学的时候，我也在上学，她成家立业时，我还在上学，她的孩子已经上幼儿园了，而我还在上学。她的一生仿佛已经过了一半，而我还在重复着这一件事情。说着说着我们都笑了。

放假，我经常会去她的店里，会经常聊天，聊到以前的事情。会讲到曾经小学男孩轰动了全校，会讲到初中坐最后一排的男生经常上课给坐第一排的她传纸条，会讲到她生大病的时候，一个男同事经常来照顾她。

只是最后她选择的，是一个通过别人介绍，见过几次就确定关系的男人。一个月后，我参加了她们的婚礼，成为了亲手为她提箱子的人。

订婚前夕，我记得我问过她是怎么想的，可是答案我已经模糊不清了。

结婚是什么？在去取火车票的路上我问她。

她说就是相夫教子，为生计而活。

生活似乎没有那么简单，婚姻也是如此。

店面的生意并不是很好，房贷的压力，店面房租的压力，养孩的压力，深深压在两个还是90后的人身上。

爱情因为婚姻，被套上了枷锁。

谈心的时候，她经常跟我讲关于她的爱情观，婚姻观。她总说一定要找一个有钱的人，生活才不会那么累。

只是我突然想到那个七夕节，

我问她七夕打算怎么过？

“有什么好过的，天天都在过。”

或许，

生活的压力就像苦瓜的味道

苦涩留于唇齿间，

只是流进腹中，才能感受到甜味。

生活的小滋味可能就是这样吧！

我所期待的爱情并不是默默在一起，直到面目全非。而是像流转的四季，每一个微妙的变化都充满了惊喜。

你将是我的新郎

发表于 2016-09-30 | 更新于: 2016-09-30 | | 热度: ℃
字数统计: 1.5k | 阅读时长 ≈ 4 
那是一张挂在橱窗里的照片，照片黑白颜色，而且破旧泛黄。照片里的男子骨骼精瘦，面容英俊。女子脸型圆润，满面春风。小小的照片停留在硕大的橱窗里，很不好看。
每天放学，走到这里，我都会放慢脚步，然后驻足停留片刻，呆呆的望着眼前的这个橱窗。

“它为什么会在这里？”
而我没有勇气踏进这家照相馆，去过问它的故事。
疑问一直深藏在心里。

大学毕业，我拖着行李箱，又回到了这座小城市，再次走到橱窗前，海滩清风、俊男美女，唯美的婚纱照占据了整个橱窗。

是的，那张照片没有了。

我终于走进了这家照相馆，然后在这里工作。

老板是一位40左右的中年妇女，一个人经营着这家照相馆。老板人很好，我叫她张姐。无论是工作上还是生活中，她都一直很关照我，她说学生不容易。

一步步走来，转而已过四个春秋。

风景如画，美女如云。每天与各色各样的人接触。记忆里，却时常让我想起那个微胖女孩，那个笑起来像花一样的美丽的女子。

“你为什么会来我这家小店呢？”有一天张姐问我。

“因为它对我有个特殊的意义。”我回答到。

“张姐，你知道以前，橱窗上有张老照片吗？”我看着她。

张姐没有回答，表情木讷，于是我也不好意思再去追问。那就让这个疑问成为秘密吧。

生活一直这样，工作顺利，生活平淡如水，波澜不惊。却在一天深夜，从母亲那里收到外婆病重的消息。

于是我订了第二天最早的飞机。整个夜晚我都在候机厅待着，记忆空白，不知所云。

“张姐，外婆病重，我回老家了。我的工作，你暂时安排别人吧，对不起。”我拿起手机，向张姐发了一条短信。

收到的只有两个字
“等你”。

病床上，外婆安睡。我轻轻的走在病床前。不知已经过了多少年，记忆里的外婆头发还是黑色，可是现已满头白发。

在记忆里，外婆永远停留在那天，我们穿着母亲寄来的新衣服，在院子里照相。“咔嚓”快门声响，新衣服的标签都未来得及剪去。可是，后来我们都离开了外婆，离开了家乡。

这好像是我和外婆唯一的一张照片吧。

眼前的外婆，就像不谙世事的婴孩，静静的睡着。

“我想我可能不会回来了。”
许久，张姐没有回应。

一个月后，外婆康复，我和外婆回到了老房子。
“妹子，这里有你的一封信。”我接过信件，寄信人赫然写着张姐的名字。
“请问什么时候寄来的？”
“嗯…差不多两个星期前”老乡想了想。
“哦，谢谢”
我将外婆搀扶到椅子上，安顿好一切，然后打开了那封信。

“关于那张照片，我一直欠你一个回答，我想给你讲个故事。”
我坐在椅子上，继续读着。

那张老照片，是我的母亲和父亲的结婚照。是父亲留给母亲唯一念想。
他们是在一家照相馆相遇的。那天是母亲的二十岁生辰，她在镜子前打扮了许久，刻意扎上了新买的红头绳。而父亲是镇里年轻有为的照相师。

“咔嚓”
母亲的微笑便留在了父亲的照相机里，于是他们相爱了。
父亲每天都会给母亲写信，然后在下午将信纸放在照相馆的窗口。母亲每天赶集回来都会从窗口取走信纸。

日久生情，情深心动。

母亲终于回了第一封信“我们结婚吧”于是便有了那张结婚照。
没有父母之命，没有媒妁之言。没有婚礼，照片就是他们爱的唯一见证。”可是不久后，父亲就没了音讯，什么都没有留下，只留下那张照片。

随后，照相馆倒闭了。母亲成了年纪轻轻的“寡妇”，还没有领结婚证的“寡妇”，然后始终未嫁。
不久后，母亲收养了我，开了一家照相馆。可能母亲希望我能像父亲一样成为一名照相师。
母亲将结婚照挂放在橱窗里，嘴角上扬，很是好看。
“妈妈，那个男的是谁？”那时我才10岁，我指着橱窗问她。她说“那是你的父亲”
“为什么要把它挂在这里呀”

“因为这样你的父亲就知道我们在这里啊”
将照片放在橱窗里，就是能够让他还能找到她，让他知道她还在等他。

只是几十年过去了，等到的是母亲病危。留下的最后一句话便是“把照片取下来吧”
她盼了一辈子，等了一辈子，然后终于累了，抱着照片睡了。

那张结婚照已经和母亲一起入土了。
母亲没了，老照片也没了，但是故事还在。溪水常流，真情难留。

珍惜。
张姐。

我合上信，多年的疑问解开，但心中还是哽咽难平。
“玲玲，帮我倒杯水。”
“哦，好的外婆”
我将杯子端到外婆的手里。

阳光美好，轻吻着外婆的脸颊，春风悄然吹过，吹落所有难言的愁绪。看着眼前水田间鸭子戏水，外婆笑了。

我望着外婆，也笑了。

我想外婆年轻的时候也应该很漂亮吧。

写给少年的她

发表于 2016-09-22 | 更新于: 2016-09-22 | 分类于 随笔 | | 热度: ℃
字数统计: 490 | 阅读时长 ≈ 1

今天，他25岁生日。在他身边的只有一个蛋糕，和一支蜡烛。关掉了灯光，他从抽屉里拿出那封信，看着窗外繁华的夜景，他想起了那个女孩。只是什么都变了，身边的景，身边的人。

他记得，十八岁那天，那个女孩一直跟在他的身后，在那个转角，女孩递了封信给他，然后女孩就逃之夭夭。

他记得，那封信是他收到的十八岁的第一份礼物，也是他收到的十八年以来的第一份礼物。

那个夏天，空气燥热，烦乱着知了。拿到那封信时，他显得没有那么意外，看着女孩逃跑的背影，他笑了，眼角泛着晶莹。

曾几何时，他会经过女孩班的后门，看到她熟睡的模样。

曾几何时，他会看到坐在末班车上的她望着他，眼神交汇，然后互相躲闪。

曾几何时，看到她跟在自己的后面，他会故意放慢脚步。

曾几何时……

后来，他为了自己所谓的理想独自北上，租着小小的房子，过着紧巴的生活。遭受过同事的白眼，遭受过老板的谩骂。

不过，他遇见了另一个她，在工作上他们互相关照，在生活中他们互相扶持。她会为他做好一日三餐，照顾好他的每一个细节。

可是，对他来说，最初那股悸动已经没有了。

再遇到那个女孩，是在那次同学聚会。当他知道她和那个女孩是一个班时，他没有一丝犹豫，毅然答应去参加聚会，只不过是以她的男友的身份出席。看见那个女孩，他露出了笑容，那个笑容简单纯粹，没有一丝杂质，只是因为他看见了那个女孩，那个，久违的朋友。

写给少年的他

发表于 2016-09-21 | 更新于: 2016-09-21 | 分类于 随笔 | | 热度: ℃
字数统计: 537 | 阅读时长 ≈ 1

车窗前的风景掠过，人影渐渐模糊，迟迟没有等到那个熟悉的身影。一次又一次，这个等待不知道重复了多久。每天坐着末班车，每天靠着车窗边，只为能够匆匆地看他一眼。一年一年，公车售票的阿姨还在，什么都没变。唯一变的，是他去了另一个城市。

她记得，他放学后总会独自一人走那条僻静的小路，而她有时也会悄悄地跟在他的身后。树影婆娑，清风掠过，唯一能听到的，只有树叶的哗啦，还有两个人的脚步声。

那个夏天，空气中弥漫着离别的味道，在那个即将分别的转角，她将那封早已写好的信递到他的手里，没有说一句话，然后仓惶离去。她不知道他收到信后是怎样的反应，诧异？嘲笑？还是不屑。她只是想表达她的单纯的喜欢，仅此而已。

夏过，转秋，悲凉而又萧瑟，小径铺满了落叶，变得冷清。他走后的生活依旧平常，没有波澜。但她还是会在某个清冷的深夜蜷缩在窗边，独自望着窗外，不说一句话。

后来，她遇到了另一个他，那个他很贴心，每天早上都会买好豆浆油条送到她的宿舍门口，中午上课前会在她的桌上放一瓶水，生病时，他会送她去医院。尽管他做的多么完美，她还是忘不了原来的那个他。忽然想到，她好像从来没有为原来的那个他做过什么。

没想到，再遇见他，是在七年后的同学聚会。他以班长的男友出席了这场聚会，他挽着班长的手，向她打招呼。酒窝浅浅，眉眼弯弯，少年依旧那么帅气，那么美好。即使七年已去，成为她夫。　　

她曾说，如果再遇见他，她想问他，你还记得那封信吗？还记得那个姑娘吗？

可是，她什么也没说。
